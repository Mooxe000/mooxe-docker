# extra

- Chisel
  - Chisel（Constructing Hardware In a Scala Embedded Language）是 UC Berkeley 开发的一种开源硬件构造语言。
    它是建构在Scala语言之上的领域专用语言（DSL），支持高度参数化的硬件生成器。
  - https://www.chiselchina.com/
